Installation

caution

This page is a work in progress.

Install TVA, Pluralsight's headless UI framework.

TVA is available as a set of npm packages for web and our tokens package is compatable for iOS, Swift, Android, and Kotlin.

npm#

For the entire kitchen sink up front, just install the components and icons packages.

  • npm
  • Yarn
npm install @pluralsight/tva-components @pluralsight/tva-icons
yarn add @pluralsight/tva-components @pluralsight/tva-icons

Or if you want maximum flexibility, just install the tokens package:

  • npm
  • Yarn
npm install @pluralsight/tva-tokens
yarn add @pluralsight/tva-tokens

PS TT Commons font#

TVA was designed with PS TT Commons in mind. So be sure to follow the typography instructions. For the best results, install via an HTML link tag:

<link
rel="stylesheet"
href="https://...TBD"
/>

Each "component" uses a variable font declaration to provide all weights programmatically. IE 11 does not support variable fonts, so any IE users will get the sans-serif fallback.

Font icons#

To use the font Icon component, you must first add the TVA icons font. For the best results, install via a HTML link tag:

<link
rel="stylesheet"
href="https://...TBD"
/>

SVG icons#

In order to use prebuilt SVG TVA icons, such as those found in the icons demos you must first install the @pluralsight/tva-icons package:

  • npm
  • Yarn
npm install @pluralsight/tva-icons
yarn add @pluralsight/tva-icons

Mobile Assets#

Our tokens package will also deliver all of our icons and fonts via assets for each mobile platform we support. ❤️

Design resources#

A set of reusable components for design tools is available, designed to match the web components and to help you craft great products:

  • Figma: A large UI kit with over 62 handcrafted TVA components.
