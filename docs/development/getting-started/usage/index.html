Usage

Learn about the use case for each package we offer.

Design Tokens‚Äã

This package is meant for one purpose - deliver a solution that has the most flexibility there is to offer for UI development.

Design tokens comes with a few things at the foundational level: our normalize CSS Reset, our font declarations, and our Design-tokens and custom themes.

If you want full control of your UI and just need a lightweight and reliable custom theming solution, Design Tokens has you covered.

  • npm
  • Yarn
npm install @pluralsight/design-tokens
yarn add @pluralsight/design-tokens

Headless Styles‚Äã

Headless Styles is built with the intention of allowing the most flexibilty in a component or native project base. This means we give you helper functions that will return an Object which delivers styles and a11y properties.

This means that you have full control of your component design without having to worry about the "annoying bits" (i.e. styles and a11y).

  • npm
  • Yarn
npm install @pluralsight/headless-styles
yarn add @pluralsight/headless-styles

Button Example‚Äã

Here's a quick example using React to create a custom Button:

components/CancelButton.jsx
import { getButtonProps } from '@pluralsight/headless-styles

const cancelBtnProps = getButtonProps({
  sentiment: 'danger'
})

export default function CancelButton(props) {
  const { children, ...btnProps } = props

  return (
    <button {...btnProps} {...cancelBtnProps.button}>
      {children}
    </button>
  )
}

The above code will render a fully styled and accessible Button when combined with our normalize setup.

Icons‚Äã

Our Icons package was created to deliver icons for a wide range of use cases: Plain SVG, React, or Svelte components. This package can also be combined with Headless-styles if you want to build components that use an enhanced visual exprience.

  • npm
  • Yarn
npm install @pluralsight/icons
yarn add @pluralsight/icons

React Utils‚Äã

This utility package is just for the React users who are looking to reduce their codebase and help make their components more accessible. In this library, we deliver custom hooks that do anything from enhancing a modal experience to better image loading.

  • npm
  • Yarn
npm install @pluralsight/react-utils
yarn add @pluralsight/react-utils
note

React-utils is compatible with React 18 and concurrent mode. üòé
